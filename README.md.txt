
#Plantilla de Wordpress de prueba

Esta plantilla tiene como objetivo enriquecer todos los
conocmientos relacionados con GitHub.

##Instalación

1.Desargar WAMP Server para ejecutarlo de manera local, e instalarlo.

2.Descargar el archivo del repositorio para poder descomprimirlo detro de un carpeta personalizada en la siguiente ruta:

###Disco local (C:)/Wamp64/www/carpeta_personalizada

3.Una vez descommprimido ejecutar Wamp Server desde el acceso directo del menú de inicio.

4.En el navegador, dirijirse a su servidor local e ingresar a la carpeta que se creo previamente, en este caso:

###localhost/carpeta_personalizada/

Ya podrá visualizar una clonación del repositorio en su computadora de manera local.

##Modificación de la plantilla

1.En el navegador, dirijirse a su servidor local e ingresar a la carpeta que se creo previamente, y luego al administrador
 proporcionado por Wordpress:

###localhost/carpeta_personalizada/wp-admin/
